# gotoiptv-gotoplayer

Carpeta reservada para fuentes enviadas desde el proyecto `gotoiptv/gotoplayer`.

Cada fuente debe entrar en una subcarpeta propia con:

- `source/`: archivos editables o fuente original.
- `notes/`: descripcion, cambios, origen y responsable.
- `license-evidence/`: evidencia de autoria, permisos, contrato o registro.
