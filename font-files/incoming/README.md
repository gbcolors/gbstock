# incoming

Colocar aqui fuentes recibidas desde otros proyectos.

Formato recomendado:

```text
incoming/
  gotoiptv-gotoplayer/
    nombre-original-recibido/
      source/
      notes/
      license-evidence/
```

Antes de publicar en GBstock:

1. Revisar autoria y origen.
2. Renombrar bajo marca `gotofont`.
3. Evitar similitud nominal con fuentes comerciales de terceros.
4. Compilar OTF, TTF y WOFF2.
5. Crear manifest y licencias.
